引入“突发性”（Burstiness）检测：

原理：AI 生成的文本通常句子长度比较平均，节奏单一。而人类写作的句子长短不一，节奏感强。

算法：计算段落中“句子长度的标准差”。如果标准差高（长短句交替），认为这是人类特征，强行扣除 AI 分数。

加大“疑罪从无”力度：

温度系数：从 1.8 提升到 2.0。

指数惩罚：从 2.5 次方提升到 3.5 次方。这样就意味着如果模型只有 80% 的把握，经过计算后会被压到 45% 左右（判定为人类）。只有模型有 95% 以上把握时，才会判定为 AI。

标点丰富度奖励：人类写作往往会使用更丰富的标点（如括号、分号、破折号），而 AI 倾向于使用逗号和句号。如果检测到丰富标点，给予“人类加分”。
