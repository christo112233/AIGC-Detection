# 🔥 AIGC大模型检测工具（本地部署版）
> 免费！中文优化版AIGC检测，精度逼近知网，毕业论文投稿前的AI率初检神器

## 🌟 核心优势
-  中文深度适配：基于AIGC_detector_zhv3模型优化，对中文论文检测更精准；
-  分段AI率展示：可显示论文每段的AI生成概率，定位疑似AI内容；
-  完全免费：无付费门槛，无需登录，本地部署不泄露论文内容；
-  小白友好：提供一键运行版本，无需编程基础，双击即可使用。

## 📋 前置要求（必看！）
本地部署核心依赖GPU，缺一不可：
1. 电脑需配备NVIDIA GPU（AMD显卡暂不支持）；
2. 已安装CUDA驱动（建议11.7及以上版本）；
3. GPU显存≥2GB（显存越大，检测速度越快）；
4. 若无GPU：仅能使用Release版本的CPU轻量化模式（速度较慢，仍可检测）。

## 🚀 快速使用（推荐）
无需配置环境，下载即用，推荐使用此方案，Releases提供的都是稳定版，功能可能滞后仓库开源版本，仓库主文件只有更新累计到一定程度时才会更新Relsases
1. 下载成品包：点击仓库右侧「Releases」，下载最新版本的压缩包（自带完整模型）；
2. 解压运行：将压缩包解压到任意文件夹（建议解压到桌面，方便查找）；
3. 启动检测：双击解压后的「AIGC Text.exe」程序，直接使用：
   - 方式1：复制粘贴论文文本到界面；
   - 方式2：直接拖入论文文档（支持docx/txt格式）；
   - 点击左下角检测按钮，等待几秒~几分钟（视论文长度），即可查看分段AI率结果。
